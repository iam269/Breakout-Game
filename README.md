## Features

- Responsive canvas that adapts to different screen sizes

- Ball physics with wall, paddle, and brick collisions
- Score tracking system
- Rules popup for game instructions
- Gradient background with custom CSS variables
- Game reset when ball is missed

## How to Play

1. Use the **Left Arrow** or **Right Arrow** keys to move the paddle horizontally.
2. Bounce the ball off the paddle to hit and destroy the bricks above.
3. Each brick destroyed increases your score.
4. If you miss the ball with the paddle, the score resets and all bricks reappear.
5. Clear all bricks to reset the level and continue playing.

## Technologies Used

- **HTML5**: Structure and Canvas element
- **CSS3**: Styling, animations, and responsive design

## Installation and Running

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. The game will start automatically.

No additional dependencies or server setup required - it's a pure client-side application.

## Game Controls

- **Left Arrow** / **Right Arrow**: Move paddle left/right
- **Show Rules** button: Display game instructions
